Project Development:


To develop this image recognition chatbot we have used these frameworks and models:
• Programming Language: Python
• GUI Framework - CustomTkinter: A modern user friendly UI library with cool graphics, fully open
source and free
• Model - LLaVA: This pre-trained model is free and open source, and is fully customizable. It comes in
3 variants. All versions use different datasets The LLaVA model, being open-source, allows for
extensive customization and scalability. The project addresses a clear need for efficient image
recognition and interaction, making it a valuable tool for various
applications, from educational purposes to professional use.
• • PyInstaller: To convert the python app to direct executable (.exe and more)
